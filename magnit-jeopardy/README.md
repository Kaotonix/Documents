# Magnit Jeopardy (static web app)

A lightweight Jeopardy-style game board with Magnit-inspired branding and clues based on publicly available content from [Magnit Global](https://magnitglobal.com/).

## Run it

- Open `magnit-jeopardy/index.html` in your browser (double-click works).

No build step. No dependencies.

## How to customize questions

Edit `questions.js`.

- The game uses 6 categories × 5 clues each.
- Each clue should have: `value` (200–1000), `clue`, `answer`.
